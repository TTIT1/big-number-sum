# Big Number Sum

Project Java thực hiện phép cộng hai số rất lớn bằng String.

## Công nghệ sử dụng
- Java
- Maven
- JUnit (Unit Test)

## Cách chạy project

Clone project:

git clone https://github.com/yourname/big-number-sum

Build project:

mvn clean install

## Chạy Unit Test

mvn test

## Ví dụ

Input:
1234 + 5678

Output:
6912
